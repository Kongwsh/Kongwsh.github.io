各种数组去重

发表于 2016-11-19   |  

要求：

  • 1 和 “1” 是不同的元素；
  • 1 和 new Number(1) 是不同的元素；
  • {} 和 {} 是不同的元素（引用不同）。

方法一

定义一个数组 res 保存结果，遍历需要去重的数组，如果该元素已经存在在 res 中了，则说明是重复的元素，如果没有，则放入 res 中。

两个for循环，复杂度为o(n^2)。

变形一

Array.prototype.indexOf()返回给定元素能找在数组中找到的第一个索引值，否则返回-1。

变形二

Array.prototype.filter() filter 为数组中的每个元素调用一次 callback 函数，并利用所有使得 callback 返回 true 或 等价于 true 的值 的元素创建一个新数组。

方法二

将原数组中重复元素的最后一个元素放入结果数组中。

方法三（sort）

将数组用 sort 排序后相同的元素会被放在相邻的位置，那么比较前后位置的元素就可以了。1 和 “1” 会被排在一起，不同的 Object 会被排在一起，因为它们 toString() 的结果相同。

方法四(对象键值对法)

hasOwnProperty():检测一个属性是否存在于实例中，继承自Object。

变形一（*）

因为 Object 的 key 值都是 String 类型，所以对于 1 和 “1” 无法分别，稍微改进下，将类型也存入 key 中。
for循环时，每次取出一个元素与对象进行对比，如果这个元素不重复，则把它存放到结果数组中，同时把这个元素的内容作为对象的一个属性，并赋值为1，存入到hash对象中。

变形二

Object.keys() 方法返回一个所有元素为字符串的数组，其元素来自于从给定的对象上面可直接枚举的属性。这些属性的顺序与手动遍历该对象属性时的一致。

方法五（ES6）（*）

ES6提供了新的数据结构Set。它类似于数组，但是成员的值都是唯一的，没有重复的值。
Array.from() 方法可以将一个类数组对象或可遍历对象转换成真正的数组。

浏览器模式（Browser Mode）

浏览器模式（Browser Mode），用于切换IE针对该网页的默认文档模式、对不同版本浏览器的条件注释解析、决定请求头里userAgent的值。它在浏览器发出请求之前就已经确定，网站没有办法修改这个值。它代表的是用户以何种浏览器访问网站。
IE11已经改名为“用户代理字符串”，其实是用来设置navigator.userAgent和navigator.appVersion的。
