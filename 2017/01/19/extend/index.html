$.extend()源码学习

发表于 2017-01-19   |     |   view times

jQuery.extend():将object1,objectN两个或更多对象的内容合并到第一个target对象。
用法：

  1. jQuery.extend( target [, object1 ] [, objectN ] )
  2. jQuery.extend( true, target, object1 [, objectN ] )
    第一个参数如果是true，合并成为递归（又叫做深拷贝）。
