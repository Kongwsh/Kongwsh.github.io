小菜鸟前端入坑记

发表于 2016-11-15   |     |   view times

近日，开始找工作，准备面试，面试受挫，感慨太多，趁机对过去的自己做个总结吧。曾经对自己有过质疑，有过痛苦因de不了的bug，但这是现在仍然前行的我亲自熬制的特色鸡汤，酌情品尝诸君。

面试小记

简历几乎石沉大海，真是令人沮丧，去几个公司面试，还是培训结构，看来我还是太弱啊。已经毕业，技术不到家，还是个女生注定没优势，回家继续修炼，再择良辰吉日出关吧（无奈.jpg）,用厉害的demo狂甩面试官的脸（滑稽.jpg）。

我为什么选择前端

大学时期，患有非典型型青春期焦虑症的我，课余时间不是思考一些形而上的问题，就是看电影，逛豆瓣，曾想当个写作者的我只写过日记。这样务虚不务实，直到大三才对学业重视，奈何出来混迟早是要还的，曾经落下的，早晚都得补回来，在MOOC上刷只在考前几周才认真学的专业课，进入大四，怀揣着对知识的渴望，加入考研大军，虽败了（悲伤.jpg），但是这段时间是我最快乐的时间，终于摆脱原地踏步，行为瘫痪(苦笑.jpg)。

程序员的世界是美好的，大家有资源会共享，有问题都会积极帮忙，用代码改变世界就算了，但是能改变自己。
是码代码的成就感，还是对于本科计算机但没写过几行代码的我想跻身程序员之列，前端的门还好进一些，还是就业的压力让我选择前端，我想都有，不然我不会坚持这么久。
既然是我自己要跳进这个坑，就不能轻易地爬出来。

愚以为生活最好的状态是好奇，热情且独立做自己喜欢的事。我不会告诉你，我追寻的是宇宙生命及一切的终极答案，哈哈哈^_~。
我不知道以后会发生什么，也许会找到更喜欢的事，但此阶段Just do it！跑起来再说。

学习总结

前端入门的我学了什么，自学难免走弯路，学习难免会枯燥

  1. 搭建个人博客
  2. 掌握JavaScript面向对象 ，CSS3，HTML5
  3. 能够编写语义化HTML，模块化CSS
  4. 响应式布局
  5. 类库jQuery
  6. Ajax，跨域JSONP,CORS
  7. 原生js实现常见交互特效，常用组件
  8. Web Audio制作音乐可视化播放web应用
  9. 熟练使用谷歌火狐调试工具
  10. 基本SEO
  11. git

我如何学习

自学可以自主把握获取知识的速度和节奏。

  1. 善用搜索引擎
    自从用了谷歌，解决问题的效率高了很多，科学上网不解释。
    除此之外，知乎，segmentfault等上的大牛有很多学习资源，感谢大家分享精神。
    无障碍访问谷歌
  2. 获得一手知识
    对于获得的解决方法或者相关知识要刨根问底，不能总是拿来主义，浅尝辄止，当然时间有限，要分清什么问题要追根溯源，什么问题要一带而过。查文档，看书，拿起高程
  3. 不想做全栈的程序员不是好前端
    做一个合格的前端，我认为，首先要是个合格的程序员，计算机的知识从最低层01到最高层的面向用户的应用程序，都应该了解。MOOC
  4. 英语要好，不然怎么看文档。扇贝
  5. 快速进步捷径是模仿，不断踩坑，光说不练假把式。DEMO

学习计划

今天开始的学习计划

  1. 组件化，框架React + ES6
  2. 构建工具gulp + webpack
  3. 后端Node.js 和 python，独立搭建起Web应用的运行环境
  4. 模块化，Require.js
  5. CSS预处理器SASS和后处理器autoprefixer
  6. 客户端/服务器端缓存，CSS/JS代码优化一类的性能方面知识
  7. js模块开发
  8. 算法知识
  9. Http和Https协议
  10. 坚持更新学习笔记

聪明伶俐，眼疾手快的我没问题，忍不住又夸自己;-)
